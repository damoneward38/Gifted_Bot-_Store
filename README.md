# Gifted Eternity Platform

A full-stack platform for music sales, AI bots, subscriptions, and creator monetization.

## Features
- Gospel music marketplace ($1/song)
- AI bot subscriptions
- Knowledge base training
- Artist dashboards
- Monetization & payouts
- User engagement tracking

## Tech Stack
- Next.js
- tRPC
- Drizzle ORM
- PostgreSQL
- PayPal SDK

## Setup
```bash
pnpm install
pnpm db:push
npm run build
npm run test
